# HTML Rules and Scope for Designing

**HCI Lab 01**  
Submitted by: Anum Rafiq

## Purpose of this Lab
This repository demonstrates the **rules and scope** of HTML in User Interface design.

## Key Learning Points

- HTML is only responsible for **Structure** and **Semantics**.
- It follows the principle of **Separation of Concerns**:
  - **HTML** → Structure (What the content is)
  - **CSS** → Presentation/Style (How it looks)
  - **JavaScript** → Behavior (How it works)

## HTML Rules Applied
- Proper document structure with `<!DOCTYPE html>`
- Use of semantic HTML tags (`<header>`, `<main>`, `<section>`, `<footer>`)
- Correct meta tags and responsive viewport
- Proper heading hierarchy
- Clean, readable, and accessible code
- Meaningful page title
